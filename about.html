1. Introduction à Javascript

C'est quoi le JavaScript ?

Le JavaScript(JS) est un langage de programmation qui donne de l'interactivité

  • HTML = structure ou le squelette
  • CSS = style (les couleurs, la mise en page etc)
  • JS = Logique (les réactions à un utilisateurs, des calculs, dynamique de la page)

Où s'éxecute le JavaScript ?

  • Dans le navigateur (Chrome, Firefox, Safari etc)
  • Sur un serveur qui tourne avec Node.js -> il ne sera plus un langage front end mais back end comme le PHP ou le Python etc...

A retenir: même langage, mais deux environnements différents.

Premiers pas : console, balise script

  1. Ouvrir votre navigateur préféré
  2. Click droit -> inspecter, ou si vous êtes sur windows F12
  3. Aller dans l'onglet "console"
